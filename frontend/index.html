    ℹ️ About This Project

    AI Legislative Analyzer is a capstone project that makes Indian parliamentary bills understandable to ordinary citizens. It uses token compression to shrink 100 000-token legal documents before running any LLM — maximising civic value while minimising energy consumption and carbon output.

    Built by Akagami (compression · RAG · LLM), rishi (ingestion · OCR · NER), and AkashSamuel (API · frontend · async queue).
